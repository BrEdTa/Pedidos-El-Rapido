# Pedidos El Rapido

App web para confirmación de pedidos con logo personalizado.

## Despliegue en Netlify

### Método 1: Arrastrar y Soltar (Más rápido)

1. Ve a [netlify.com](https://www.netlify.com) y crea una cuenta (es gratis)
2. Una vez dentro del panel, busca la opción **"Add new site"** → **"Deploy manually"**
3. Arrastra la carpeta del proyecto o el archivo `index.html` directamente
4. ¡Listo! Netlify te dará una URL como: `https://random-name-123.netlify.app`
5. Abre esa URL en tu iPhone desde Safari

### Método 2: Con Git (Recomendado para actualizaciones)

1. Crea un repositorio en GitHub (puedes hacerlo desde [github.com](https://github.com))
2. Sube tu proyecto al repositorio
3. En Netlify, ve a **"Add new site"** → **"Import an existing project"**
4. Conecta tu cuenta de GitHub y selecciona el repositorio
5. Netlify detectará automáticamente que es un sitio estático
6. Haz clic en **"Deploy site"**
7. Cada vez que hagas cambios y los subas a GitHub, Netlify actualizará automáticamente

### Acceder desde iPhone

- Simplemente abre Safari en tu iPhone
- Escribe la URL que Netlify te proporcionó
- También puedes agregarlo a la pantalla de inicio como si fuera una app nativa:
  - En Safari, toca el botón de compartir (cuadrado con flecha)
  - Selecciona "Agregar a pantalla de inicio"
  - ¡Ya tienes tu app en el iPhone!
